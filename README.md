# Weather App MVP

Cloud-only MVP untuk eksplorasi data prakiraan cuaca menggunakan OpenWeather API.

## Stack
- Python
- Streamlit Community Cloud
- OpenWeather API (Free Tier)

## Environment Variables
Disimpan di Streamlit Secrets:
- OPENWEATHER_API_KEY
